XtalComp: Compare crystal structures

XtalComp Web Interface: Enter POSCAR files

Paste a POSCAR file into each of the following boxes and hit "Compare" to let XtalComp determine if they describe the same structure.

Limitations:

  • The POSCARs must specify fractional ("Direct") coordinates. The XtalComp CGI wrapper does not currently check/convert from cartesian units.
  • Each structure must have the same number of atoms, and the order of the atom type counts (line 6) must be the same in each POSCAR.

About:

XtalComp is an algorithm developed by David C. Lonie for the genetic algorithm XtalOpt. It will determine if two crystal structure descriptions refer to the same underlying structure, despite choice of unit cell vectors, rotations, reflections, translations, and numerical noise. It is soon to be released under the "New" BSD open source license and described fully in an academic journal. Check back for updates as the XtalComp library approaches its release.
